Typography

Vitre is designed for ordinary paragraphs, inline links, emphasized text, strong text, marked text, and small metadata such as May 1, 2026.

Abbreviations like CSS, inserted text like new token names, deleted text like old aliases, and formulas like H2O or x2 should sit cleanly in running copy.

Small text should stay legible without competing with body copy.

    Button Variables

    Default buttons use a tokenized gradient, hover border, glow, shadow, and scale. The second button disables only the background image through a CSS variable. 

    :root {
  --vitre-button-bg-angle: 180deg;
}

<button type="button" data-vitre-gradient="none">Flat button</button>
<button type="button" data-vitre-glow="none">No glow</button>
